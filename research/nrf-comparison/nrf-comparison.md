# Create me a comparison tool for nRF chips, using them as microcontrollers, microprocessors, or just for Bluetooth alongside another microcontroller or microprocessor

I've created an interactive nRF chip comparison tool for you. Here's what it includes:

**Three viewing modes:**

- **Card View** - Individual chip specifications in a clean card format
- **Table View** - Full comparison table with all specs
- **Selection Matrix** - Quick visual scoring for power efficiency, performance, memory, and protocol support

**Smart filtering:**

- Filter by use case (Standalone SoC, BLE + External MCU, Simple Beacon, ML/High-Performance)
- Filter by memory requirements (< 100 KB, 100-500 KB, > 500 KB Flash)
- Filter by clock speed (64 MHz, 128 MHz, 320 MHz+)

**Full coverage of current nRF lineup:**

- **nRF52 Series** (52805-52840): Mature, proven, Cortex-M4, good for existing designs
- **nRF5340**: Dual-core powerhouse for complex multi-protocol apps
- **nRF54L Series** (05, 10, 15): New generation with RRAM, 30% better power efficiency, Bluetooth 6.0
- **nRF54H20**: High-performance flagship with 320 MHz, dual cores + RISC-V, ML support

**Key features highlighted:**

- Current consumption profiles
- TX/RX power specs for Bluetooth range
- Protocol support (BLE, Thread, Matter, Zigbee, NFC, Mesh)
- Memory architecture (flash, RAM, cache)
- Temperature range and supply voltage

You can export filtered results as CSV for further analysis. The tool supports dark mode and is fully responsive for mobile viewing.
