# Build me an interactive comparison tool for the following ESP32 variants and any others you can find information on.

Xtensa: ESP32, ESP32-S2, ESP32-S3, ESP32-U4 (original and "S" variants)
RISC-V: ESP32-C2, ESP32-C3, ESP32-C6, ESP32-H2 ("C" and "H" variants)

I want to compare features, peripherals, cores, speed, anything you can think of. Xtensa cores (listed above) should be tagged a different colour from RISC-V cores (also listed above).

I've created an interactive ESP32 comparison tool with all the variants you mentioned plus a few additional ones. Here's what you get:

**Architecture Tagging:** Xtensa cores are highlighted in warm orange (ESP32, S2, S3, U4), RISC-V cores in cool blue (C2, C3, C5, C6, H2, H6, P4).

**Key Features:**

- **Search**: Filter by chip name or features in real-time
- **Architecture Filter**: Toggle between All, Xtensa, and RISC-V views
- **Sortable Columns**: Click headers to sort by chip name, core count, or frequency
- **Comprehensive Specs**: CPU architecture, cores, frequency, memory (SRAM/ROM/Flash), GPIO count, wireless connectivity, and distinctive features

**Notable Variants Included:**

- **Xtensa Line**: Original ESP32 (dual-core, Swiss Army knife), S2 (USB OTG beast), S3 (AI accelerator with most GPIO), U4 (antenna-ready S3 variant)
- **RISC-V Line**: C2 (ultra-compact budget option), C3 (sweet spot single-core), C5 (dual-band WiFi future), C6 (WiFi 6 smart home powerhouse), H2 (thread/zigbee mesh without WiFi), H6 (industrial powerhouse), P4 (fastest—400MHz but no wireless)

The tool respects your system's dark/light mode preference and is fully responsive for mobile viewing. Perfect for quickly comparing specs when designing your next IoT project.
