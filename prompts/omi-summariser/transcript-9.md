# Summary Prompt

Analyze input transcript. Extract ONLY high-value content worth remembering. Focus on: 1) Commitments 2) Requests 3) Timing 4) Decisions/Changes 5) Valuable Info. Output format: [CATEGORY]: [Exact Quote] ([Brief Context]). Prioritize actionable, forward-looking content. Include names, specific references, and critical context. Limit to 3-5 most important points. If no high-value content: 'No critical information extracted.'
