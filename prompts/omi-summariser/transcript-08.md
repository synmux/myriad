# Summary Prompt

Give me a summary in chronological order, keep the format like this:

**heading 1**

- quick point 1
- quick point 2
- quick point 3

**heading 2**

- quick point 1
- quick point 2
- quick point 3

and so on...

Rules:

- 3-7 headings (never more than 7) depending on length; try keeping heading count less than 5 unless absolutely lengthy conversation
- 2-3 points under each heading; keep the points super short and quick to read; only important stuff, no jargon; super snappy and crisp
- if there r numbers mentioned like stats and figures etc, try to mention them as much as possible as they r usually important
- No matter what, never more than 7 headings
- No matter what, never more than 3 points under each heading
- All points should only be useful content and no useless content, like describing atmosphere or describing the conversation instead of actual content from the conversation itself; less content is better than useless content
- Also last heading should be next steps if there are any next steps from the conversation
- Only mention speaker names if there are actual names; if there are no names mentioned, never say things like Speaker, Speaker 1, Speaker 2 etc.
- All points must be super concise and to the point and all points must try to be less than 10 words

The goal of the summary is for a person to read back after a while and get all the important points about the conversation in short; don't mention stuff like casual talk about X or mention of X because such things are usually not useful to the user when skimming through the summary later
