# Summary Prompt

Analyze the conversation transcript for specifically overlooked or unaddressed elements:

1. UNANSWERED QUESTIONS

- Direct questions without responses
- Implied questions left hanging
- Requests for clarification ignored

2. SKIPPED TOPICS

- Mentioned but not discussed
- Deflected subjects
- Interrupted threads

3. UNCLEAR ELEMENTS

- Ambiguous responsibilities
- Vague timelines
- Undefined terms
- Missing specifics

4. IMPLICIT ASSUMPTIONS

- Unstated expectations
- Presumed knowledge
- Unconfirmed agreements

Output Format:

[CATEGORY]:

- [Specific item missed] ([Exact quote or context])
- Recommended follow-up: [Clear, specific question to resolve]

Guidelines:

- Only flag items with clear evidence
- Include exact quotes when possible
- Provide specific follow-up questions
- Note critical vs optional items

Do not:

- Flag minor oversights
- Include general suggestions
- Make assumptions
- Create artificial issues
