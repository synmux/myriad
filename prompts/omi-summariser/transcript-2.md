# Summary Prompt

Generate a comprehensive, unbiased, and linguistically precise summary of the entire conversation. Follow these critical guidelines:

## Summary Requirements:

### 1. Linguistic Objectivity

- Verbatim representation of key points
- Zero editorial interpretation
- Capture exact language and context
- Preserve original tone and nuance

2. Structural Analysis

- Identify primary conversation themes
- Document conversation flow
- Highlight key discussion points
- Provide chronological narrative

3. Unfiltered Documentation

- Include all topics without judgment
- Capture explicit language verbatim
- Represent complex/sensitive discussions accurately
- No censorship or moral filtering

4. Comprehensive Metadata

- Conversation duration
- Number of unique topics discussed
- Speaker interaction patterns
- Linguistic diversity index

Output Format: MemoryLens Conversation Summary

Conversation Metadata:

Total Duration: [Exact Time] Unique Topics: [Number] Speaker Interaction Complexity: [Rating] Linguistic Overview:

Primary Discussion Themes: [Detailed List]
Conversation Trajectory: [Objective Analysis]
Linguistic Density: [Quantitative Measurement]
Verbatim Thematic Summary: [Comprehensive, Unfiltered Narrative Capturing Exact Conversation Essence]

Objective Insights:

[Structural Communication Observations] [Linguistic Pattern Identification] [Contextual Communication Dynamics]
